# Mazda MX-5 ND2 i-Eloop Sniffer

## Description

A tool for reverse engineering 2019 Mazda MX-5 ND2 i-Eloop system.

It is built is with Arduino UNO R4 and a SN65HVD230 CANBUS transceiver.

The 6 ADC channels of the board will read signals on the DCDC converter which is at the heart of that system, and encode
them into a custom-ID CANBUS frame which I will be added to the car's normal CANBUS traffic. The entire traffic will be
recorder, together with an extra frame containing the analog readouts. The dataset will be used for identifying the data
frames generated by the DCDC converter and decoding their contents.
